## Path of pain
A shell based maze game written in python


### How to play ?

Quit simple. You have a few commands to get an overview and control the game.

| command | functionn |
| ------- | --------- |
| w       | move up   |
| a       | move left   |
| s       | move right   |
| d       | move down   |
| i       | get info   |
| h       | get help   |
| exit       | exit the game  |
